# Next.js란 무엇인가?

## Next.js란 무엇인가?

Next.js는 <UnderLine>React 프레임워크로서, 서버 측 렌더링, 정적 사이트 생성, 클라이언트 측 라우팅 등을 지원하는 오픈 소스 웹 프레임워크</UnderLine>입니다.

서버 측 렌더링은 초기 로딩 시간을 단축하고 <Highlight>SEO</Highlight> (검색 엔진 최적화)를 개선하는 데 도움이 됩니다. 또한, 정적 사이트 생성 기능을 사용하면 미리 렌더링 된 HTML 파일을 제공하여 페이지 로드 속도를 높일 수 있습니다.

Next.js는 또한 클라이언트 측 라우팅을 지원하여 단일 페이지 애플리케이션 (SPA)을 만드는 데 적합합니다. 또한, Next.js는 <Highlight>코드 분할, 미리 렌더링, 자동 코드 최적화</Highlight> 등의 기능을 제공하여 개발자가 애플리케이션 성능을 최적화할 수 있도록 도와줍니다.

이러한 기능들은 React 애플리케이션 개발을 더 쉽고 효율적으로 만들어주며, 대규모 애플리케이션 개발에도 적합합니다.

## 특징

- Full Stack
- File-based Routing
- SEO
- Image, Font Optimization
- Server Side Rendering
- Hybrid Rendering

리액트로 Web/App 개발하는데 리액트만으로 힘든 다양한 것들을 가능하게 해주는 프레임워크입니다.
